# Treasure Slime v0.7.4

Treasure Slime is a browser-based endless action runner with idle progression elements.

## Features

- Hold to rise, release to fall
- Automatic bubble attacks
- Treasure Rat enemies
- Coins and treasure chests
- Special random chest events
- Coin Shop and Treasure Shop
- Slayer Paths
- Persistent stats and achievements
- Free slime colors and rainbow skin
- Offline coin rewards
- Mobile and keyboard controls
- Browser localStorage saving

## Controls

- Hold Space / W / Up Arrow / Tap: rise
- Release: fall
- P or Escape: pause
- Menu button: shops, skins, stats, and Slayer Paths

## Run Locally

Open `index.html` in a modern browser.

## GitHub Pages

1. Upload these files to a GitHub repository.
2. Open **Settings**.
3. Open **Pages**.
4. Set the source to the `main` branch and `/root`.
5. Save.
